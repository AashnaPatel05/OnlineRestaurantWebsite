# 🌿 Spice Garden – Online Restaurant Website

This project is a web-based online restaurant system developed as part of a group coursework. It allows users to browse menu items, add food to cart, and place orders through a simple and user-friendly interface.

The system is built using Java, JSF (Jakarta Faces), and Apache Derby database.

---

## 👨‍💻 Features

- User registration and login system
- Menu display with different categories
- Add to cart and update cart functionality
- Checkout system with payment options
- Order placement and confirmation
- Customer reviews system
- Session management (user stays logged in)
- Form validation and error handling
- AJAX functionality for dynamic updates

---

## 🛠 Technologies Used

- Java (Backend)
- JSF / Facelets (Frontend)
- Apache Derby (Database)
- HTML & CSS (UI Design)
- GitHub (Version Control)

---

## 📂 Project Structure

- `Web Pages` → XHTML pages (UI)
- `Source Packages` → Java Beans, DAO, Entities, Util
- `resources` → CSS, JavaScript, Images
- `database.sql` → Database script

---

## ▶️ How to Run the Project

1. Open project in NetBeans
2. Start Apache server (GlassFish)
3. Run the project
4. Open browser:
   http://localhost:8080/OnlineRestaurantWebsite
